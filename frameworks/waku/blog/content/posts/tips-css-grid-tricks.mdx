CSS Grid has revolutionized how we create layouts on the web. Here are some practical tips and tricks to help you master CSS Grid and create stunning, responsive designs.

## 1. The fr Unit for Flexible Columns

The `fr` (fractional) unit distributes available space proportionally:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 1rem;
}

/* First and third columns get 1 part each, middle gets 2 parts */
```

```css
/* Mix fixed and flexible columns */
.sidebar-layout {
  display: grid;
  grid-template-columns: 250px 1fr 200px;
  /* Fixed sidebar, flexible content, fixed aside */
}
```

## 2. Auto-Fit and Auto-Fill for Responsive Grids

Create responsive grids without media queries:

```css
/* Auto-fit: columns collapse when content is sparse */
.gallery-auto-fit {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}

/* Auto-fill: maintains column count even with empty space */
.gallery-auto-fill {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 1rem;
}
```

## 3. Named Grid Lines for Better Organization

Use named lines instead of numbers for clearer code:

```css
.layout {
  display: grid;
  grid-template-columns: 
    [sidebar-start] 250px 
    [sidebar-end main-start] 1fr 
    [main-end aside-start] 200px 
    [aside-end];
  grid-template-rows: 
    [header-start] 60px 
    [header-end content-start] 1fr 
    [content-end footer-start] 40px 
    [footer-end];
}

.header {
  grid-column: sidebar-start / aside-end;
  grid-row: header-start / header-end;
}

.main {
  grid-column: main-start / main-end;
  grid-row: content-start / content-end;
}
```

## 4. Grid Template Areas for Visual Layout

Create layouts that mirror your visual design:

```css
.page-layout {
  display: grid;
  grid-template-areas: 
    "header header header"
    "sidebar main aside"
    "footer footer footer";
  grid-template-columns: 250px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

/* Responsive breakpoint */
@media (max-width: 768px) {
  .page-layout {
    grid-template-areas: 
      "header"
      "main"
      "sidebar"
      "aside"
      "footer";
    grid-template-columns: 1fr;
  }
}
```

## 5. Overlapping Items with Grid

Create interesting overlapping layouts:

```css
.hero {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 1fr 1fr;
}

.hero-image {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
  z-index: 1;
}

.hero-text {
  grid-column: 1 / 2;
  grid-row: 2 / 3;
  z-index: 2;
  background: rgba(255, 255, 255, 0.9);
  padding: 2rem;
}
```

## 6. Subgrid for Nested Layouts

Use subgrid to align nested items with parent grid:

```css
.card-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

.card {
  display: grid;
  grid-template-rows: subgrid;
  grid-row: span 3;
}

.card-title { grid-row: 1; }
.card-content { grid-row: 2; }
.card-footer { grid-row: 3; }
```

## 7. Centering Content with Grid

The easiest way to center content:

```css
.center-container {
  display: grid;
  place-items: center;
  min-height: 100vh;
}

/* Or use place-content for container alignment */
.center-content {
  display: grid;
  place-content: center;
  min-height: 50vh;
}
```

## 8. Grid Inspector in DevTools

**Chrome/Edge DevTools:**
- Select grid container
- Click grid badge next to element
- Use overlay to visualize grid lines and areas

**Firefox DevTools:**
- Built-in grid inspector
- Shows line numbers, area names, and dimensions
- Excellent for debugging complex layouts

## 9. Masonry-like Layout

Create Pinterest-style layouts:

```css
.masonry {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  grid-auto-rows: 10px;
  gap: 10px;
}

.masonry-item {
  grid-row-end: span var(--row-span);
}
```

```javascript
// Calculate row span based on content height
const items = document.querySelectorAll('.masonry-item');
const rowHeight = 10;

items.forEach(item => {
  const height = item.offsetHeight;
  const rowSpan = Math.ceil(height / rowHeight);
  item.style.setProperty('--row-span', rowSpan);
});
```

## 10. Grid Animation Tips

Animate grid properties smoothly:

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  transition: grid-template-columns 0.3s ease;
}

.grid-container.expanded {
  grid-template-columns: repeat(4, 1fr);
}

/* Animate individual items */
.grid-item {
  transition: grid-column 0.3s ease, grid-row 0.3s ease;
}

.grid-item.featured {
  grid-column: span 2;
  grid-row: span 2;
}
```
