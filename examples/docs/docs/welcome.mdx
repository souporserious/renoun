Get started with this example and learn core features: sidebar navigation, quick links, and MDX-powered content.

Use the sidebar to explore the documentation.

## Key Features

<Accordion title="Sidebar Navigation">
  The sidebar provides easy access to all documentation sections. It's
  automatically generated from your MDX files and supports nested navigation.
</Accordion>

<Accordion title="MDX Support">
  Write your documentation using MDX, which allows you to use React components
  directly in your markdown content. This enables rich, interactive
  documentation.
</Accordion>

<Accordion title="Quick Links">
  Quick links help users navigate between related documentation sections.
  They're automatically generated based on your documentation structure.
</Accordion>
