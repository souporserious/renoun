Master your workflow with our comprehensive set of keyboard shortcuts. These shortcuts are designed to help you navigate and use the editor efficiently.

## Editor Navigation

- `Cmd/Ctrl + P`: Quick file navigation
- `Cmd/Ctrl + Shift + P`: Command palette
- `Cmd/Ctrl + B`: Toggle sidebar
- `Cmd/Ctrl + J`: Toggle terminal
- `Cmd/Ctrl + \`: Split editor
- `Cmd/Ctrl + W`: Close editor
- `Cmd/Ctrl + Tab`: Switch between editors

## Code Navigation

- `Cmd/Ctrl + Click`: Go to definition
- `Cmd/Ctrl + Shift + F`: Search in files
- `Cmd/Ctrl + Shift + O`: Symbol search
- `Cmd/Ctrl + G`: Go to line
- `Cmd/Ctrl + U`: Go back
- `Cmd/Ctrl + Shift + U`: Go forward

## Editing

- `Cmd/Ctrl + /`: Toggle comment
- `Cmd/Ctrl + D`: Select next occurrence
- `Cmd/Ctrl + Shift + L`: Select all occurrences
- `Cmd/Ctrl + Enter`: Insert line below
- `Cmd/Ctrl + Shift + Enter`: Insert line above
- `Alt + Up/Down`: Move line up/down

## AI Features

- `Cmd/Ctrl + K`: Open AI command palette
- `Cmd/Ctrl + L`: Explain selected code
- `Cmd/Ctrl + I`: Inline code completion
- `Cmd/Ctrl + Shift + I`: Generate code from comment

## Customization

You can customize these shortcuts in the settings menu under Keyboard Shortcuts. We also support keybinding profiles for different workflows and preferences.
