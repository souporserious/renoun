# Apps

This directory contains the renoun documentation site along with the published renoun applications.

### [Blog](./blog)

A blog example showing content management and organization.

- Blog post management
- MDX support
- Custom collections
- Post organization

### [Docs](./docs)

A complete documentation site example built with renoun.

- MDX support for content
- Custom UI components
- Tailwind CSS styling
- Next.js configuration

### [Workbench](./workbench)

Design system workbench for building, previewing, and documenting libraries.

- Component library structure
- Custom hooks
- MDX documentation
- Component examples
