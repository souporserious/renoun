Omnidoc is a powerful and flexible toolkit designed to elevate your JavaScript technical content. Whether you're building a blog, writing documentation, or showcasing your design system, Omnidoc provides everything you need to create engaging, interactive, and most importantly valid content that reflects your source code.

## Features

- **File System Collections**: Organize and retrieve documentation based on your project's file structure and existing source code.
- **Type Documentation Generation**: Automatically generate detailed type documentation to keep your content accurate and up-to-date.
- **MDX and React Integrations**: Write content with a mix of Markdown and JSX using MDX plugins and React components.
- **Syntax Highlighting**: Improve code readability with accurate syntax highlighting for multiple languages.
- **Module Export Validation**: Ensure your module exports stay consistent and are correctly documented using schema validations.
- **Code Snippet Type-Checking**: Improve code reliability with type-checking for code snippets across marketing and documentation contents.
- **Framework Agnostic**: Omnidoc works with any JavaScript framework, enhancing flexibility and compatibility.
- **Optimized for performance**: Enjoy a lightweight and fast documentation process, optimized for developer efficiency.
- **Customizable**: Tailor every aspect of your documentation to fit your brand and style guidelines.

Ready to start? Check out the [Getting Started guide](/getting-started) to dive into Omnidoc and see how easy and efficient your documentation process can be!
